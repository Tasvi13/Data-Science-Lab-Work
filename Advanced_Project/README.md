# Fraud Detection (Credit Card Transactions)

## Description
This project detects fraudulent credit card transactions using machine learning classification models.

## Dataset
- Source: Kaggle (Credit Card Fraud Detection Dataset)
- Description: Contains transaction features and a target class label where 1 indicates fraud and 0 indicates normal transaction.

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Checking Class Imbalance
4. Feature Scaling
5. Model Building (Logistic Regression / Random Forest)
6. Model Evaluation using classification metrics

## Results
- Metrics Used: Accuracy, Precision, Recall, F1-score
- Confusion Matrix was used to evaluate fraud detection performance.

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Conclusion
This project demonstrates how machine learning can be used to detect fraudulent transactions and reduce financial risk.
