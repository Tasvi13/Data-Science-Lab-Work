# Data Science Lab Work

## Description
This repository contains my Data Science Lab work, including Python programming basics, NumPy, Pandas, Regression, KNN Classification, and Machine Learning projects.

It is organized into lab folders and project folders as per the course requirements.

---

## Folder Structure

### Labs
- **Lab1_Introduction_to_Python**  
  Basics of Python programming, variables, data types, and operators.

- **Lab2_Control_Statements**  
  Programs using if-else, loops, factorial, prime, palindrome, etc.

- **Lab3_NumPy**  
  NumPy array creation, slicing, reshaping, and mathematical operations.

- **Lab4_Pandas**  
  DataFrames, missing values handling, filtering, sorting, and groupby operations.

- **Lab5_Regression_Analysis**  
  Linear Regression model building and evaluation.

- **Lab6_KNN_Classification**  
  KNN classification using the Iris dataset with scaling and evaluation.

---

### Projects
- **Fun_Project**  
  Simple Python-based fun project (example: Rock Paper Scissors game).

- **Intermediate_Project**  
  Machine Learning intermediate level project (example: House Price Prediction).

- **Advanced_Project** *(Optional)*  
  Advanced level machine learning project (example: Fraud Detection / Sentiment Analysis).
