# Rock Paper Scissors Game

## Description
This is a simple Rock Paper Scissors game where the user plays against the computer.
The computer randomly selects rock, paper, or scissors and the winner is decided based on game rules.

## Dataset
- No dataset is used.
- This is a Python-based interactive game.

## Steps Performed
1. Took user input (rock/paper/scissors)
2. Generated computer choice randomly
3. Compared both choices using conditions
4. Displayed winner of each round
5. Stored and displayed score until the user quits

## Results
- The program displays the winner of each round.
- It also displays the final score when the user exits.

## Tools Used
- Python
- random module

## Conclusion
This project demonstrates the use of conditional statements, loops, and random number generation in Python.
