# House Price Prediction

## Description
This project predicts house prices using machine learning regression models.
It includes EDA, preprocessing, feature engineering, and model training.

## Dataset
- Source: Kaggle (House Prices: Advanced Regression Techniques)
- Description: Dataset contains house features like area, garage size, basement area, etc., and the target column is SalePrice.

## Steps Performed
1. Exploratory Data Analysis (EDA)
2. Missing Value Handling
3. Numerical and Categorical Feature Analysis
4. Feature Engineering and Feature Creation
5. Model Training and Testing
6. Regression Models (Lasso, Ridge, XGBoost)
7. Final Submission File Generation

## Results
- Models Used: Lasso Regression, Ridge Regression, XGB Regressor
- Evaluation Metrics: RMSE / R2 Score

## Tools Used
- Python
- NumPy, Pandas
- Matplotlib, Seaborn
- Scikit-learn
- XGBoost

## Conclusion
The project successfully analyzes house data, performs preprocessing, and builds regression models to predict house prices with good accuracy.
